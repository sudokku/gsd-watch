Create `internal/parser/cache.go` in package `parser`:

1. Struct definition:
```go
type ProjectCache struct {
    root   string                // path to .planning/ directory
    data   ProjectData           // cached project data
    mtimes map[string]time.Time  // path -> last known mtime
}
```

2. Constructor: `func NewCache(root string) *ProjectCache` — returns `&ProjectCache{root: root, mtimes: make(map[string]time.Time)}`

3. `func (c *ProjectCache) ParseFull() ProjectData`:
   - Calls `c.data = ParseProject(c.root)` (reuses existing function)
   - Walks `c.root` recursively, populating `c.mtimes` for every file: `filepath.WalkDir(c.root, func(path string, d fs.DirEntry, err error) error { if err == nil && !d.IsDir() { if info, statErr := d.Info(); statErr == nil { c.mtimes[path] = info.ModTime() } }; return nil })`
   - Returns `c.data`

4. `func (c *ProjectCache) Update(path string) ProjectData`:
   a. **Mtime guard**: `info, err := os.Stat(path)` — if err != nil (file deleted), fall through to ParseFull(). If `c.mtimes[path].Equal(info.ModTime())`, return `c.data` unchanged. Otherwise update `c.mtimes[path] = info.ModTime()`.
   b. **Path routing** using `filepath.Base(path)` and `filepath.Rel(c.root, path)`:
      - `filepath.Base(path) == "STATE.md"`: call `st, err := parseState(path)`. If err == nil, update `c.data.CurrentAction = st.StoppedAt` (or "unknown" if empty), `c.data.ProgressPercent = float64(st.ProgressPercent) / 100.0`, `c.data.Name = st.MilestoneName` (if non-empty). Then re-parse phases to update active markers: `phaseNames := parseRoadmap(filepath.Join(c.root, "ROADMAP.md"))`, `c.data.Phases = parsePhases(filepath.Join(c.root, "phases"), phaseNames, st.ActivePhase, st.ActivePlan)`.
      - `filepath.Base(path) == "config.json"`: call `cfg, err := parseConfig(path)`. If err == nil, update `c.data.ModelProfile = cfg.ModelProfile`, `c.data.Mode = cfg.Mode`.
      - `filepath.Base(path) == "ROADMAP.md"`: call `c.data = ParseProject(c.root)` (roadmap changes affect phase names globally — full re-parse is simplest and correct).
      - Path matches `planFileRe` (NN-NN-PLAN.md) or ends in `-SUMMARY.md`: extract phase dir from path (`filepath.Dir(path)`), find matching phase in `c.data.Phases` by `DirName == filepath.Base(phaseDir)`, get the phase dir's parent phase number from `phaseDirRe`, re-parse plans for that phase: `plans := parsePlansInDir(phaseDir, phaseNum, activePhase, activePlan)` where activePhase/activePlan come from parsing STATE.md (call parseState or use cached values). Update `c.data.Phases[i].Plans = plans` and re-derive phase status with `derivePhaseStatus(plans)`.
      - Path matches a badge file suffix (CONTEXT.md, RESEARCH.md, VERIFICATION.md, UAT.md but NOT planFileRe): extract phase dir, find matching phase, re-detect badges for that phase by iterating `badgeFiles` and checking `os.Stat`.
      - **Default (unknown path)**: call `c.data = ParseProject(c.root)` (safe fallback).
   c. Update `c.data.LastUpdated = time.Now()` on any successful update.
   d. Return `c.data`.

5. Helper to extract active phase/plan from cached state for plan re-parsing: parse STATE.md from cache root (`filepath.Join(c.root, "STATE.md")`).
