Create `internal/watcher/watcher.go` with:

1. Package declaration: `package watcher`

2. Import: `fsnotify`, `filepath`, `io/fs`, `os`, `sync`, `time`, and `github.com/radu/gsd-watch/internal/tui` (for `tui.FileChangedMsg`)

3. Function signature: `func Run(root string, events chan<- tea.Msg)` — runs forever, meant to be called as `go watcher.Run(...)`. The `root` parameter is the `.planning/` directory path. The `events` parameter is a send-only channel of `tea.Msg`.

4. Inside `Run()`:
   a. Create fsnotify watcher: `w, err := fsnotify.NewWatcher()` — if err, log to stderr and return
   b. `defer w.Close()`
   c. Walk root recursively to add all directories:
      ```go
      filepath.WalkDir(root, func(path string, d fs.DirEntry, err error) error {
          if err != nil { return nil } // skip unreadable, don't abort
          if d.IsDir() { return w.Add(path) }
          return nil
      })
      ```
   d. Debounce state: `var mu sync.Mutex` and `timers := make(map[string]*time.Timer)`
   e. Event loop with `for { select { ... } }`:
      - `case e, ok := <-w.Events:` — if !ok, return
        - Filter: `if !e.Has(fsnotify.Write) && !e.Has(fsnotify.Create) { continue }`
        - Dynamic dir add (WATCH-02): `if e.Has(fsnotify.Create) { if info, err := os.Stat(e.Name); err == nil && info.IsDir() { _ = w.Add(e.Name) } }`
        - Debounce (WATCH-03): capture `path := e.Name` locally, then:
          ```go
          mu.Lock()
          t, exists := timers[path]
          if !exists {
              t = time.AfterFunc(math.MaxInt64, func() {
                  events <- tui.FileChangedMsg{Path: path}
                  mu.Lock()
                  delete(timers, path)
                  mu.Unlock()
              })
              t.Stop()
              timers[path] = t
          }
          mu.Unlock()
          t.Reset(300 * time.Millisecond)
          ```
      - `case err, ok := <-w.Errors:` — if !ok, return; else log to stderr with `fmt.Fprintf(os.Stderr, "watcher error: %v\n", err)` and continue

5. Import `math` for `math.MaxInt64`, `fmt` for stderr logging.
