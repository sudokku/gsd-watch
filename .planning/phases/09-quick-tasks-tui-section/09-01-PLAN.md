    **Step 3: Create internal/parser/quick.go.**

    Package `parser`. Imports: `os`, `path/filepath`, `regexp`, `sort`, `strings`.

    Define:
    ```go
    // quickTaskDirRe matches YYMMDD-ID-description subdirectory names.
    var quickTaskDirRe = regexp.MustCompile(`^(\d{6})-(\w+)-(.+)$`)
    ```

    Implement `func parseQuickTasks(quickDir string) []QuickTask`:
    - Call `os.ReadDir(quickDir)`. On error, return nil (dir absent per D-02).
    - Iterate entries. Skip non-directories (`!entry.IsDir()`).
    - Match entry name against `quickTaskDirRe`. Skip non-matching.
    - Extract captures: `date = m[1]`, `id = m[2]`, `rest = m[3]`.
    - `displayName = strings.ReplaceAll(rest, "-", " ")`
    - Build file paths: `planFile = date + "-" + id + "-PLAN.md"`, `summaryFile = date + "-" + id + "-SUMMARY.md"`
    - Check with `os.Stat(filepath.Join(taskDir, summaryFile))` and `os.Stat(filepath.Join(taskDir, planFile))`.
    - Status logic:
      - `hasSummary == nil` -> `StatusComplete`
      - `hasPlan == nil` -> `StatusInProgress`
      - else -> `StatusPending`
    - Emit `debugf("quick_task_dir", "%s status=%q display=%q", entry.Name(), status, displayName)`.
    - Append QuickTask to slice.
    - After loop: `sort.Slice(tasks, func(i, j int) bool { return tasks[i].Date > tasks[j].Date })` — newest first.
    - Return tasks.
